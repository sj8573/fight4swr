🏠 導論：準實驗設計 (Quasi-Experimental Design)

在真實的社會工作實務現場，我們很難做到像實驗室一樣的「隨機分派」(Random Assignment)。

例如：

  • 倫理問題：你不能「隨機」指派一半的家暴受害者「不准」接受服務（作為控制組）。
  • 實務問題：你不能「隨機」告訴一半的案主「你不能」參加這個方案，因為他們是控制組。
  • 行政問題：機構的服務對象通常是「自行上門」或「轉介而來」，而不是隨機分派的。

當「隨機分派」不可行時，「真實驗設計」就無法使用。這時，「準實驗設計」就成了我們在實務現場評估方案成效、建立「近似」因果關係的最佳選擇。

「準」(Quasi) 的意思就是「近似」、「好像」。準實驗設計就是「近似」真實驗的設計，它通常具備「介入 (X)」和「控制組」，但最大的特徵就是**缺乏「隨機分派 (R)」**。

本章節學習重點：

本章節將介紹幾種最強大、最常見的準實驗設計，並探討當我們「無法」隨機分派時，會面臨哪些「額外」的效度威脅，以及我們該如何補強我們的因果推論。

⚖️ 準實驗設計類型

準實驗設計的目標，就是試圖在「不能隨機分派」的限制下，盡可能地「模擬」出一個嚴謹的比較基礎，以排除「選擇偏誤」等威脅。

符號說明：

  • O: 觀察 / 測量 (Observation)
  • X: 介入 / 實驗處理 (Treatment)
  • N: 非隨機分派的組別 (Nonequivalent Groups)
  • (時間由左至右)

1. 不等控制組前後測設計 (Nonequivalent Control Group Design)

N O₁ X O₂
N O₁ O₂

這是「最常見」也「最強大」的準實驗設計。它看起來和「真實驗」的前後測控制組設計一模一樣，唯一的差別就是「沒有 R」（隨機分派）。

研究者使用「既有的」兩組人（如：A 學校的學生、B 學校的學生），一組當實驗組，一組當控制組。兩組都做前測 (O₁) 和後測 (O₂)。

關鍵：雖然沒有隨機分派，但「前測 (O₁)」非常重要。我們可以比較兩組的「前測分數」，如果 O₁ 分數「相近」，我們就能「更有信心」地說兩組在後測 (O₂) 的差異是「介入 (X)」造成的。如果 O₁ 分數「差異很大」，則因果推論就非常薄弱。

社工範例：

研究問題：評估「方案 A」對提升 A 學校學生的「情緒控管能力」是否有效。
設計：A 學校（實驗組）接受方案 A (X)，B 學校（控制組）沒有。在方案開始前 (O₁)，先測量「兩間學校」學生的情緒控管能力；結束後 (O₂)，再測一次。
分析：比較兩校的「進步幅度」(O₂ - O₁)。

2. 時間序列設計 (Time-Series Design)

O₁ O₂ O₃ X O₄ O₅ O₆

當「找不到」控制組時，可以使用這種設計。研究者對「同一組」對象，在「介入 (X) 之前」進行「多次」的觀察（O₁, O₂, O₃），在「介入 (X) 之後」也進行「多次」的觀察（O₄, O₅, O₆）。

關鍵：多次的前測可以建立一個「基準線」(Baseline)。如果介入「真的有效」，我們應該會看到在 X 發生之後，趨勢線發生了「明顯的轉折」。

優點：能有效排除「成熟」或「統計迴歸」（因為基準線會顯示出原本的趨勢）。 缺點：無法排除「歷史事件」（如果剛好在 X 和 O₄ 之間發生了某個大事）。

社工範例：

研究問題：評估一個「新的社區警示系統 (X)」是否能「降低」社區的「犯罪率」。
設計：不找控制組。而是去調閱該社區「過去三年」每個月的犯罪率 (O₁, O₂...O₃₆)。在「裝設系統 (X)」之後，繼續追蹤「未來三年」每個月的犯罪率 (O₃₇...O₇₂)。
分析：觀察裝設 (X) 之後的犯罪率，是否「顯著低於」原本的趨勢線。

📚 隨堂測驗：準實驗設計類型

🔬 其它對效度的威脅

當研究「缺乏隨機分派」（如準實驗設計）或參與者「知道自己正在被研究」時，會產生一些額外的內在效度威脅，這些威脅大多與「人的心理」有關：

1. 霍桑效應 (Hawthorne Effect)

定義：當參與者「知道」自己正在「被觀察」或「被研究」時，他們的行為會「自動」變得更積極、更正向，無論介入 (X) 本身是否有效。

威脅：你會誤以為是「介入 (X)」有效，但實際上只是「被關注」的效果。

社工範例：

社工想測試一個「新督導方法 (X)」。實驗組的社工「知道」自己正在參與研究，因此「特別努力」工作，導致績效提升。但這可能不是新方法有效，而是「霍桑效應」造成的。

2. 安慰劑效應 (Placebo Effect)

定義：參與者因為「相信」自己正在接受一個「有效」的治療，僅憑此「信念」就導致了症狀的改善，即使他們拿到的只是「無效」的安慰劑（如糖果）。

威脅：你會誤以為是「介入 (X)」有效，但實際上只是「心理預期」的效果。

社工範例：

在評估一個「減壓團體」時，實驗組的成員因為「相信」這個團體很有名、很有效，光是「參加」這件事就讓他們覺得壓力減輕了。

3. 約翰亨利效應 (John Henry Effect)

定義：「控制組」的成員「知道」自己正在和「實驗組」比較，為了「不認輸」或「證明自己不差」，他們反而「異常努力」表現，導致其表現高於平常水準。

威脅：控制組的表現「異常地好」，會導致你「低估」介入 (X) 的真實效果（因為兩組差距變小了）。

社工範例：

A 班（實驗組）接受新教學法，B 班（控制組）沒有。B 班學生「得知」此事後，為了「證明」自己不比 A 班差，反而「更拚命」讀書，導致期末兩班成績「沒有顯著差異」。

4. 研究對象士氣低落 (Demoralization)

定義：「控制組」的成員「知道」自己「沒有」被分到「比較好」的實驗組，感到「失望」、「被剝奪」或「放棄」，因此表現得「比平常更差」。

威脅：控制組的表現「異常地差」，會導致你「高估」介入 (X) 的真實效果（因為兩組差距變大了）。

社工範例：

研究一個「新的」戒癮藥物。控制組的成員得知自己拿到的是「安慰劑」後，感到「絕望」，於是「放棄」戒癮，導致其復發率「異常地高」。

📚 隨堂測驗：其它對效度的威脅

情境：在一個方案評估中，控制組的成員發現實驗組「每週」都可以拿到「很棒的」新資源，而自己什麼都沒有，因此感到非常「失望」，並放棄了原本的努力。

這會導致研究者「高估」方案的效果。這種威脅稱為什麼？

📊 實驗與準實驗的比較

真實驗設計與準實驗設計是社工研究中，最常用於「評估方案成效」的兩種「量化」途徑。它們的核心差異在於「隨機分派」：

比較面向 🔬 (真)實驗設計 ⚖️ 準實驗設計
關鍵特徵 ✅ 必須有「隨機分派」(R) ❌ 缺乏「隨機分派」(R)
控制組 必須有（或多個）控制組 「通常」有控制組（或使用基準線）
內在效度 高
（能有效排除「選擇偏誤」等威脅）		 中等
（「選擇偏誤」是最大的威脅）
外在效度 低
（實驗室情境、嚴格控制，可能不貼近真實世界）		 較高
（在「真實實務現場」進行，情境較自然）
因果推論 強
（建立因果關係的黃金標準）		 中等
（只能提供「近似」的因果證據）
實務可行性 低
（倫理和實務上通常不可行）		 高
（符合實務現場的限制，最常被使用）
社工範例 隨機指派案主接受 A 療法或 B 療法。 比較 A 機構（有執行方案）與 B 機構（沒執行）的案主差異。
