🏠 導論：尋找因果

在社會工作中，我們最關心的問題之一就是：「我的介入『是否』真的『造成』了案主的改變？」換句話說，我們渴望建立「因果關係」(Causality)。

例如，我們想知道：

  • 「親職教育團體」是否「導致」家長管教技巧的提升？
  • 「就業輔導方案」是否「造成」失業者找到工作的機率增加？
  • 「某種心理治療」是否「引起」案主憂鬱症狀的減輕？

然而，「建立因果關係」遠比「描述相關性」要困難得多。僅僅觀察到 A 和 B 一起發生，並不代表 A 就是 B 的原因。例如，冰淇淋銷量和溺水人數高度相關，但不是吃冰淇淋「導致」溺水，而是因為夏天（第三變項）同時導致了兩者。

「實驗設計」(Experimental Design) 被認為是建立因果關係「最強而有力」的方法，因為它透過「操弄」自變項與「隨機分派」，最大限度地排除了其他可能的解釋。

本章節學習重點：

本章節將帶您了解建立因果關係的「三大必要條件」，區分研究的「內在效度」與「外在效度」，認識「威脅」因果推論的常見因素，並介紹幾種常用（但強度不同）的實驗設計類型。

🔑 因果關係的條件

要宣稱「X 導致了 Y」，必須「同時」滿足以下三個條件：

1️⃣ 相關性 (Correlation / Association)

X 的變化必須與 Y 的變化「有所關聯」。當 X 改變時，Y 也會跟著改變（無論是正相關或負相關）。如果 X 和 Y 之間完全沒有統計上的關聯，那就不可能存在因果關係。

社工範例：

如果你想證明「參與讀書會 (X)」能「提升學業成績 (Y)」，你必須先證明「參與時數」和「成績分數」之間，確實存在「正相關」。如果兩者根本沒關聯，那讀書會就不可能是原因。

2️⃣ 時間順序 (Time Order / Temporal Precedence)

「原因」必須發生在「結果」之前。X 必須先於 Y 發生。這是最直觀但也最關鍵的條件。

社工範例：

要證明「接受心理治療 (X)」能「減輕憂鬱 (Y)」，你必須證明案主是「先」接受了治療，「之後」憂鬱才減輕。如果是憂鬱「先」自己好轉，案主「才」去接受治療，那就不是治療的效果了。

3️⃣ 非虛假關係 (Non-spuriousness)

X 與 Y 之間的「觀察到的相關性」，不能是由「第三個變項 Z」(Lurking Variable) 所「同時導致」的「假象」。必須排除掉所有「其他可能的解釋」。這是最難達成的條件。

社工範例：

觀察：研究發現「居住在擁擠社區 (X)」與「青少年犯罪率 (Y)」呈現正相關。
可能的虛假關係：也許不是「擁擠」本身導致犯罪，而是因為「貧窮 (Z)」同時導致了人們「只能住在擁擠社區」和「犯罪率較高」。
如何排除：研究者必須在統計上「控制」貧窮這個變項，看看在「相同貧窮程度」下，擁擠與犯罪是否「仍然」相關。

📚 隨堂測驗：因果關係的條件

情境：研究發現，夏天時「冰淇淋的銷量」和「溺水死亡人數」都同步上升，呈現高度正相關。有人因此推論「吃冰淇淋會導致溺水」。

這個推論最可能違反了因果關係的哪一個條件？

🎯 內在效度 vs. 外在效度 (Internal vs. External Validity)

在評估一個研究（尤其是因果研究）的品質時，我們最關心兩種效度：

1. 內在效度 (Internal Validity)

定義：指研究結果的「可信度」。我們有多大的「信心」可以說，觀察到的「結果 Y」（如：憂鬱減輕）「真的」是由我們的「介入 X」（如：心理治療）所「導致」的，而不是由「其他無關因素 Z」造成的？

關鍵問題：研究設計是否「嚴謹」到足以「排除」其他可能的解釋？

如何提升：

  • 使用「隨機分派」(Random Assignment)。
  • 設立「控制組」(Control Group)。
  • 控制環境因素。

社工範例：

一個設計「嚴謹」的藥物實驗（有隨機分派、雙盲、安慰劑控制組），其「內在效度」通常很高。我們可以很有信心地說，療效是藥物造成的。

2. 外在效度 (External Validity)

定義：指研究結果的「可推論性」或「普遍性」(Generalizability)。我們有多大的「信心」可以將這個研究的發現，「推論」到「研究樣本以外」的「其他群體」、「其他情境」或「其他時間」？

關鍵問題：研究的「樣本」是否具有「代表性」？研究的「情境」是否「貼近真實世界」？

如何提升：

  • 使用「隨機抽樣」(Random Sampling)。
  • 確保樣本能代表母群體。
  • 在「真實世界」（而非實驗室）進行研究。

社工範例：

一個只在「某間大學」對「心理系學生」進行的實驗，其「外在效度」可能很低。我們很難將結果推論到「所有大學生」，更不用說「社會大眾」。

效度的權衡 (Trade-off)

內在效度與外在效度之間，常常存在「權衡關係」。

  • 實驗室實驗：通常「內在效度」很高（控制嚴謹），但「外在效度」可能很低（情境不真實、樣本特殊）。
  • 真實世界調查：通常「外在效度」較高（樣本代表性好、情境真實），但「內在效度」較低（難以控制變項、建立因果）。

研究者必須根據「研究目的」，來決定要優先確保哪種效度。

📚 隨堂測驗：內在 vs. 外在效度

情境：一位研究者想知道某個「新設計的網站介面」是否真的比較好用。他只找了 10 位「自己系上的同學」來實驗室「模擬操作」，並比較新舊介面的差異。

雖然他在實驗室中控制得很好（內在效度可能不錯），但我們很難將這個結果「推論」到「一般的網路使用者」。這主要是因為這個研究在哪方面的效度可能較低？

⚠️ 威脅內在效度的因素 (Threats to Internal Validity)

當我們觀察到一個「改變」（如：方案介入後案主進步了）時，除了我們的「介入」之外，還有許多「其他的可能性」也可能造成這個改變。這些「其他可能性」就是「內在效度的威脅」，我們必須盡力排除它們：

1. 歷史事件 (History)

在研究進行期間（前測到後測之間），發生了「外部的重大事件」，影響了結果。

社工範例：

你正在進行一個為期半年的「社區減壓方案」。期間，社區發生了「重大天災」。期末測量發現大家壓力都「升高」了，這可能不是你的方案「無效」，而是「歷史事件」造成的。

2. 成熟 (Maturation)

參與者隨著「時間自然推移」，生理或心理上產生了「自然的」變化。

社工範例：

你對一群「青少年」進行為期一年的「情緒管理團體」。一年後發現他們情緒穩定度「提升」了，這可能只是因為他們「自然長大成熟」了，不一定是團體的效果。

3. 測試 (Testing)

「前測」本身就影響了「後測」的表現（如：練習效應、回憶起答案）。

社工範例：

你用同一份「親職壓力量表」做前測和後測。家長在後測時分數「降低」了，可能只是因為他們「熟悉」了題目，而不是壓力真的減輕。

4. 工具衰變 (Instrumentation)

測量「工具」或「程序」在前測和後測之間發生了「改變」。

社工範例：

(1) 前測用 A 量表，後測改用 B 量表。(2) 前測是 A 訪員，後測換成 B 訪員（訪談風格不同）。(3) 觀察者（如：評估兒童行為）的「評分標準」隨時間改變（疲勞或學習）。

5. 統計迴歸 (Statistical Regression)

如果只挑選「極端分數」（非常高或非常低）的參與者，他們在下一次測量時，分數會傾向於「自然地」向「平均數」靠近，即使沒有任何介入。

社工範例：

你只挑選了全校「憂鬱分數最高」的 10% 學生來參加減壓團體。後測發現他們分數「顯著下降」了，這可能部分是因為「統計迴歸」效應，而不是團體真的那麼有效。

6. 選擇偏誤 (Selection Bias)

實驗組和控制組在「研究開始前」就已經存在「系統性差異」。

社工範例：

你比較「自願參加」親職團體的家長（實驗組）和「沒空參加」的家長（控制組）。結果發現實驗組進步較多，但這可能是因為他們「原本就比較有動機」，而不是團體的效果。

7. 樣本流失 (Mortality / Attrition)

在縱貫研究中，實驗組和控制組的「中途退出率」或「退出者的特質」有「系統性差異」。

社工範例：

在戒酒方案研究中，實驗組中「狀況最差」的人都中途退出了，只剩下狀況好的；而控制組都留下來了。期末比較時，實驗組看起來「效果很好」，但這是因為「樣本流失」造成的假象。

8. 介入擴散 (Diffusion / Imitation)

「控制組」的成員意外地接觸到了「實驗組」的介入內容。

社工範例：

實驗組的家長學到了新的教養技巧，回去跟控制組的朋友「分享」。結果控制組也進步了，導致兩組的「差異變小」，低估了方案的真實效果。

📚 隨堂測驗：威脅內在效度的因素

情境：一個研究者想評估「新藥物」對改善老人憂鬱的效果。他將安養院中「自願」服用新藥的老人當作實驗組，將「不願意」服藥的老人當作控制組。期末發現實驗組憂鬱分數改善較多。

這個研究設計最可能受到哪一種內在效度威脅的影響？

🧪 前實驗設計 (Pre-experimental Designs)

前實驗設計是一些「常見」但「內在效度很低」的研究設計。它們通常缺乏「隨機分派」或「控制組」（或兩者皆無），因此很難排除內在效度的威脅，無法有效建立因果關係。

雖然它們不嚴謹，但在「資源有限」或「初步探索」時，仍可能被使用。

符號說明：

  • O: 觀察 / 測量 (Observation)
  • X: 介入 / 實驗處理 (Treatment)
  • R: 隨機分派 (Random Assignment)
  • (時間由左至右)

1. 單組後測設計 (One-Shot Case Study)

X O

只對「一組」人進行「介入 (X)」，然後進行「一次後測 (O)」。

問題：完全沒有「比較」的基準。你不知道如果「沒有」介入，結果會是怎樣？

社工範例：

辦了一場「親職講座 (X)」，結束後發問卷問家長「滿意度 (O)」。就算滿意度很高，你也不知道講座是否「真的」改變了什麼。

2. 單組前後測設計 (One-Group Pretest-Posttest Design)

O₁ X O₂

對「一組」人進行「前測 (O₁)」，然後「介入 (X)」，再進行「後測 (O₂)」。比較 O₁ 和 O₂ 的差異。

問題：雖然有了「比較」（自己跟自己比），但「無法排除」歷史事件、成熟、測試、工具衰變、統計迴歸等「所有」時間相關的威脅。

社工範例：

對一群學生進行「憂鬱前測 (O₁)」，然後執行「減壓方案 (X)」，再進行「憂鬱後測 (O₂)」。就算 O₂ 分數低於 O₁，也可能是因為「期末考結束」（歷史事件）或「學生長大了」（成熟），不一定是方案的效果。

3. 靜態組比較設計 (Static-Group Comparison)

X O
O

找「兩組」人，一組接受「介入 (X)」，另一組「沒有」。然後「同時」對兩組進行「後測 (O)」，比較兩組差異。

問題：沒有「前測」，且通常「不是」隨機分派。我們「無法確定」兩組在「一開始」是否就不同（選擇偏誤）。

社工範例：

比較「有參加」課後輔導班的學生 (X+O) 和「沒參加」的學生 (O) 的「學業成績」。就算有參加的成績較好，也可能是因為他們「原本就比較積極」想學習。

📚 隨堂測驗：前實驗設計

情境：社工為一群新住民舉辦了一系列適應課程 (X)，並在課程「結束後」測量他們的社區參與度 (O)。

這種只有「介入」和「後測」的設計 (X O)，稱為什麼？

🔬 (真)實驗設計 (True Experimental Designs)

真實驗設計被視為建立因果關係的「黃金標準」(Gold Standard)。它們之所以強大，是因為具備兩個關鍵要素：

  • 1. 控制組 (Control Group)：有一組「沒有」接受實驗介入的對象，作為「比較」的基準。
  • 2. 隨機分派 (Random Assignment)：參與者被分配到「實驗組」或「控制組」是「完全隨機」的，這能確保兩組在「研究開始前」的各種特質（已知的和未知的）都是「大致相等」的，最大限度地排除了「選擇偏誤」。

注意：「隨機分派」(Random Assignment) ≠ 「隨機抽樣」(Random Sampling)。前者是為了「內在效度」（確保組別相等），後者是為了「外在效度」（確保樣本代表性）。

1. 前後測控制組設計 (Pretest-Posttest Control Group Design)

R O₁ X O₂
R O₁ O₂

最經典的設計。參與者被「隨機分派 (R)」到實驗組或控制組。兩組都接受「前測 (O₁)」。實驗組接受「介入 (X)」，控制組則無（或接受安慰劑/常規服務）。最後兩組都接受「後測 (O₂)」。

分析：比較兩組在「後測分數 (O₂)」上的差異，或比較兩組「進步幅度」(O₂ - O₁) 的差異。

優點：能有效排除大部分內在效度威脅（歷史、成熟、統計迴歸等，因為兩組都會經歷）。 缺點：前測本身可能產生「測試」效應，影響外在效度。

社工範例：

想評估「方案 A」對降低青少年攻擊行為是否有效。隨機將 100 位青少年分派到「方案 A 組」或「常規服務組」。兩組都先測攻擊行為 (O₁)。A 組接受方案 (X)，常規組照舊。三個月後再測兩組的攻擊行為 (O₂)。比較兩組 O₂ 或 (O₂-O₁) 的差異。

2. 後測控制組設計 (Posttest-Only Control Group Design)

R X O
R O

與前者類似，但「省略」了「前測 (O₁)」。參與者被「隨機分派 (R)」。實驗組接受「介入 (X)」，控制組則無。最後兩組都只接受「後測 (O)」。

分析：直接比較兩組在「後測分數 (O)」上的差異。

假設：因為有「隨機分派 (R)」，我們可以「假設」兩組在「研究開始前」是相等的，所以省略前測也沒關係。

優點：排除了「測試」效應的干擾；有時前測不可行或太貴。 缺點：無法確知兩組「起始點」是否真的完全相等；無法分析「進步幅度」。

社工範例：

想評估一個「新的」預防自殺宣導短片 (X) 是否有效。隨機將觀眾分派去看「新短片」或「無關短片」。看完後，「立刻」測量他們對求助資源的認知程度 (O)。比較兩組 O 的差異。

3. 所羅門四組設計 (Solomon Four-Group Design)

組1: R O₁ X O₂
組2: R O₁ O₂
組3: R X O₂
組4: R O₂

最嚴謹但也最複雜的設計。它結合了「前後測控制組」和「後測控制組」兩種設計，共有「四組」。

分析：可以同時比較：
- 介入效果 (組1 vs 組2；組3 vs 組4)
- 前測的影響 (組1 vs 組3；組2 vs 組4)

優點：能最完整地評估「介入效果」以及「前測本身」可能造成的干擾。內在與外在效度都最高。 缺點：非常昂貴、耗時、需要大量樣本。

社工範例：

極少在社工實務中使用，通常只在資源充足的「大型研究計畫」中才會考慮。

📚 隨堂測驗：(真)實驗設計

情境：一個研究者想驗證「新療法 X」是否比「舊療法」更有效。他將 100 位案主「隨機」分配到 A 組（接受新療法）或 B 組（接受舊療法）。三個月後，比較兩組的改善程度。

這個設計中，「隨機分配」這個步驟，最主要的目的在於提升研究的哪一種效度？
